123 - Pattern: Right Hand Triangle

123_MD_Pattern_RIGHT.js Patterns

Common matrix logic exercises construct patterns via bounded variable iteration loops mapped to dimensional length indices.

// n = 3a
// *
// * *
// * * *

let n = 3;
for (let i = 1; i <= n; i++) {
    let row = " ";
    for (let j = 1; j <= i; j++) {
        //row = row + "* ";
        row += "* ";
    }
    console.log(row.trim());
}

124 - Pattern: Inverted Left Hand Triangle

124_MD_Left_hand.js Patterns

Looping in reverse sequentially enables structuring reversed triangular matrices logic.

// *****
// ****
// ***
// **
// *

let n = 5;
for (let i = n; i >= 1; i--) {
    let row = "";
    for (let j = 1; j <= i; j++) {
        row += "*";
    }
    console.log(row);
}

125 - Pattern: Pyramid Extrusion

125_Pyramid_Pattern.js Patterns

By blending spacer increment calculations combined with star mapping constraints, complex pyramidal structures are plotted progressively.

//   *
//  ***
// ***** 

let n = 3;
for (let i = 1; i <= n; i++) {

    let row = "";
    for (let j = 1; j <= n - i; j++) {
        row += " ";
    }
    for (let j = 1; j <= 2 * i - 1; j++) {
        row += "*";
    }
    console.log(row);

}
